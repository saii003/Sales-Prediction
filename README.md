# 📈 Walmart Weekly Sales Prediction using Linear Regression

## 📌 Project Overview

This project predicts Walmart weekly sales using Machine Learning. A Linear Regression model is trained on historical sales data to estimate future weekly sales based on different input features.

---

## 🎯 Objective

Develop a predictive model that helps estimate Walmart's weekly sales accurately using historical data.

---

## 📂 Dataset

- Dataset: Walmart Data Analysis and Forecasting
- Target Variable: Weekly_Sales

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## 🤖 Machine Learning Algorithm

- Linear Regression

---

## 📊 Model Evaluation

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## 🚀 Features

- Data preprocessing
- Feature selection
- Model training
- Sales prediction
- Performance evaluation
- Visualization of Actual vs Predicted Sales
